Tab 1 — Black-Scholes-Merton
BSM Option Pricing Calculator
The Black-Scholes-Merton model gives a closed-form solution for European option prices under specific assumptions: no dividends, constant volatility, log-normal stock prices, continuous trading, no transaction costs.
BSM Call Price
C = S₀·N(d₁) − K·e−rT·N(d₂)
d₁ = [ln(S₀/K) + (r + σ²/2)T] / (σ√T)  |  d₂ = d₁ − σ√T  |  Put = C − S₀ + K·e−rT (Put-Call Parity)

BSM ASSUMPTIONS
Log-normal stock prices · Constant σ · Continuous trading · No dividends · No transaction costs · European exercise only · Risk-free borrowing
MONEYNESS IMPACT
Deep ITM: mostly intrinsic value. ATM: maximum time value. Deep OTM: mostly time value but small absolute. ATM options have highest sensitivity to σ changes (vega).
BSM LIMITATIONS
Underprices deep OTM options (volatility smile). Assumes constant σ (not true in practice). Cannot handle early exercise (American options). Dividends require modification.
Tab 2 — Binomial Tree Model
Binomial Option Pricing
The binomial model discretises time into steps. At each step, the stock can go up by factor u or down by factor d. Risk-neutral probabilities p and (1−p) are used to price the option by backward induction. Works for both European and American options.
Up/Down Factors (CRR)
u = eσ√Δt  |  d = 1/u = e−σ√Δt
Risk-neutral probability: p = (erΔt − d) / (u − d)
Backward Induction
f = e−rΔt [p·f_u + (1−p)·f_d]
American option: f = max(intrinsic value, discounted expected value)

WHEN TO USE BINOMIAL
✅ American options (early exercise possible)
✅ Dividend-paying stocks
✅ Path-dependent options
✅ When you need to see the tree (intuition)
❌ Slow for large N
❌ Not as elegant as BSM closed form
WHEN TO USE BSM
✅ European options
✅ No dividends (or known dividends)
✅ Fast calculation
✅ Greeks in closed form
❌ Cannot handle early exercise
❌ Constant volatility assumption is unrealistic

Spot Price Sensitivity
As S₀ increases: call value rises (Delta > 0), put value falls (Delta < 0). The rate of change (Delta) itself changes with S₀ (captured by Gamma). Deep ITM calls have Delta ≈ 1; deep OTM calls have Delta ≈ 0.
